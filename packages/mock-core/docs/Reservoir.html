Class: Reservoir

Reservoir()

new Reservoir()

Represents a Reservoir object that keeps track of the number of traces per second sampled and the fixed rate for a given sampling rule. This information is fetched from X-Ray serivce. It decides if a given trace should be borrowed or sampled or not sampled based on the state of current second.
